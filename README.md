Zadanie rekrutacyjne na stanowisko .net developer.

Po skończeniu pracy zauważyłem, że sposób w jaki przechowywane są typy pojazdów oraz kategorie prawa jazdy kierowców można byłoby zaimplementować w inny sposób bardziej wspierający ekspansję api. 
W tym momencie jest to trochę twardo zakodowane np: "motorcycle" albo "BD". Utworzenie tabel w bazie danych dla tych zmiennych polepszyłoby jakoś kodu, usuwając lub ułatwiając walidację ich wartości.
Starałem się używać dobrych praktyk programistycznych oraz zasad czystego kodu. Aplikacja działa zgodnie z treścią zadania, testowana w postmanie oraz swaggerze. Nie zdecydowałem się na implementacje frontendu.
